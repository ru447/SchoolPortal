# SchoolPortal

A comprehensive web-based school management system designed to simplify academic and administrative operations for students, teachers, and administrators.

## Features

* Secure authentication for Students, Teachers, and Administrators
* Student dashboard for viewing grades, results, and report cards
* Teacher dashboard for managing marks, assignments, and learning resources
* Admin dashboard for managing users, classes, subjects, and reports
* Results and grade management
* Past paper repository
* PDF document upload and download
* Assignment management
* Notifications system
* Search functionality

## Tech Stack

* **Backend:** Java, Spring Boot
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Database:** MySQL
* **Version Control:** Git & GitHub

## System Architecture

```text
Browser
   │
Frontend (HTML/CSS/JavaScript)
   │
Spring Boot Backend
   │
MySQL Database
```

## Core Modules

* Authentication & User Management

* **Phase 1:** Authentication and user dashboards
* **Phase 2:** Results, grades, and report generation
* **Phase 3:** Past papers, document sharing, and PDF management
* **Phase 4:** Notifications, messaging, and responsive design
